# Assets Directory

This directory should contain all game assets organized into subdirectories:

## Expected Structure

```
assets/
├── images/
│   ├── pokecon.png
│   ├── lab.png
│   ├── lab_op.png
│   ├── battle.jpg
│   ├── prof_sir.png
│   └── trainer_sheet.png
│
├── audio/
│   ├── Ches.mp3
│   ├── Pip.mp3
│   ├── Tep.mp3
│   ├── Encounter.mp3
│   ├── ICY Ches.mp3
│   ├── ICY Pip.mp3
│   ├── ICY Tep.mp3
│   ├── lowhp.mp3
│   ├── Button.mp3
│   ├── introvid.mp4
│   ├── Debulsan halimaw.mp3
│   └── Dialogue 1.MP3 through Dialogue 5.MP3
│
└── fonts/
    ├── Pokefont.ttf
    ├── 8-BIT WONDER.TTF
    └── Helvetica.ttf
```

## Note

Place your game assets in the appropriate subdirectories before running the game. The game will look for these files in the locations specified above.
